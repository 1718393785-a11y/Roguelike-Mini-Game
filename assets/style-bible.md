# 千里走单骑美术风格基准

## 核心风格

- 暗黑三国，俯视 45 度。
- 战场小尺寸可读性优先，轮廓高对比。
- 低饱和战场底色，武器和技能使用高亮识别色。
- 透明背景优先；生成工具不支持透明时使用纯色 chroma-key 背景再移除。
- 不生成带文字的运行时图标，文字由游戏 Canvas/UI 层绘制。

## 通用负向提示

文字，水印，背景场景，复杂构图，低清晰度，过度写实，3D 渲染感，侧视，正面肖像，表情特写，多角色混杂，裁切身体，模糊边缘，光污染遮挡主体。

## 首批接入顺序

1. 武器/技能 UI 图标。
2. 拾取物与木箱。
3. 敌人与 Boss 静态精灵。
4. 玩家精灵。
5. 战斗特效。
6. 地图 tile 与 Pixi atlas。
